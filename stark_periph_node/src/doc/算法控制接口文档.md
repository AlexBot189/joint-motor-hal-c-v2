# Stark Motor Control API

## 交付文件

- `stark_client.h` -- 所有 API, Header-Only, 零依赖
- `stark_shm.h`   -- 数据结构

## 编译

```bash
gcc -O2 your_algo.c -lpthread -lrt -lm -o your_algo
# 或 C++
g++ -O2 your_algo.cpp -lpthread -lrt -lm -o your_algo
```

## 模板 (复制即用)

```c
#include "stark_client.h"

int main() {
    stark_client_t c;

    /* 1. 连接 SHM, 等待就绪 */
    while (stark_open(&c) != 0) usleep(100000);
    while (!stark_ready(&c)) usleep(100000);

    /* 2. 使能 + 设模式 */
    stark_enable(&c, 1);  stark_enable(&c, 2);
    stark_set_mode(&c, 1, STARK_MODE_CURRENT);
    stark_set_mode(&c, 2, STARK_MODE_CURRENT);
    usleep(5000);

    /* 3. 控制循环 */
    uint32_t rpt_ver = 0, rt_cycle = 0;
    while (1) {
        motor_data_t m1 = stark_fb(&c, 1);
        motor_data_t m2 = stark_fb(&c, 2);
        imu_data_t  imu = stark_imu(&c);

        stark_multi(&c, t1, 0, 0, t2, 0, 0);  // 一发双电机
        usleep(1000);

        /* 周期上报: 每5ms有新数据 */
        const PeriodicUploadData* d;
        if (stark_report_try_read(&c, &rpt_ver, &d)) {
            // d->RealtimeVelocity, d->motor_abs_angle, d->cal_Iq_current ...
        }

        /* 每200ms: 心跳 + stark_node存活检测 */
        static int tick = 0;
        if (++tick % 200 == 0) {
            stark_heartbeat(&c);
            if (!stark_rt_alive(&c, &rt_cycle)) {
                while (!stark_ready(&c)) usleep(100000);
                stark_enable(&c, 1); stark_enable(&c, 2);
                usleep(5000);
            }
        }
    }

    stark_estop(&c, 1); stark_estop(&c, 2);
    stark_close(&c);
}
```

## API 速查

### 生命周期 + 状态

| 函数 | 说明 |
|------|------|
| `stark_open(&c)` | 连接 SHM, 0=成功. 可循环重试等 stark_node 启动 |
| `stark_close(&c)` | 断开 |
| `stark_ready(&c)` | 1=校准完成, 可控制 |
| `stark_online(&c, id)` | 1=电机在线, id: 1=右 2=左 |
| `stark_state(&c)` | 0=BOOTING 1=READY 2=RUNNING 3=FAULT |

### 控制命令 (实时, 1KHz)

```c
/* 双电机一帧 (推荐) */
stark_multi(c, t1,v1,p1, t2,v2,p2);

/* 单电机 */
stark_torque(c, id, mA);         /* 电流模式  */ 
stark_speed(c, id, rpm);         /* CSV 速度   */
stark_position(c, id, deg);      /* CSP 位置   */
stark_pp(c, id, deg, acc, vel);  /* PP 轮廓位置 */
stark_pv(c, id, rpm, acc);       /* PV 轮廓速度 */
stark_mit(c, id, pos,vel, kp,kd,tor);  /* MIT 阻抗 */
```

`stark_multi` 参数含义跟随当前模式: CURRENT 只用 t1/t2, CSP 用 p1/p2, CSV/PV 用 v1/v2.

### 管理命令

```c
stark_enable(c, id);        // 使能
stark_disable(c, id);       // 失能
stark_estop(c, id);         // 急停
stark_set_mode(c, id, mode);// 切换模式
stark_clear_fault(c, id);   // 清故障
```

管理命令和控制命令间隔至少 5ms.

### 反馈 (零拷贝)

```c
motor_data_t  fb = stark_fb(&c, id);
/* fb.position(编码器, 转角度: deg=pos*360/65536), fb.velocity(RPM),
   fb.current_iq(mA), fb.temperature(0.1C) */

imu_data_t  imu = stark_imu(&c);
/* imu.roll/pitch/yaw(deg), imu.acc_x/y/z(g), imu.gyro_x/y/z(dps) */
```

### 周期上报 (校准后自动开启)

```c
uint32_t ver = 0;
const PeriodicUploadData* d;
if (stark_report_try_read(&c, &ver, &d)) {
    // 电机: d->RealtimeVelocity(RPM*10), d->motor_abs_angle(deg*10),
    //       d->cal_Iq_current(A*100),  d->motor_temp(C*100)
    // IMU:  d->gyro_roll/pitch/yaw, d->quat_w/x/y/z, d->acc_x/y/z
    // 传感器: d->hall_a/b/c, d->df181_torque, d->knee_angle
}
```

### 心跳

```c
stark_heartbeat(&c);                    // 声明存活, 每200ms
stark_rt_alive(&c, &last_cycle);        // stark_node存活, 0=需重连
```

## 控制模式

| 常量 | 说明 |
|------|------|
| STARK_MODE_CURRENT (5) | Q轴电流直控 (外骨骼推荐) |
| STARK_MODE_CSP (3) | 循环同步位置 |
| STARK_MODE_CSV (4) | 循环同步速度 |
| STARK_MODE_PP (1) | 轮廓位置 |
| STARK_MODE_PV (2) | 轮廓速度 |
| STARK_MODE_MIT (6) | MIT 阻抗控制 |

## 电机 ID

- 1: 右髋
- 2: 左髋

## 规则

1. root 权限运行
2. 管理命令和控制命令间隔 ≥5ms
3. 双电机用 stark_multi
4. 每200ms调 stark_heartbeat + stark_rt_alive
5. 外骨骼: 不发命令时反馈正常, 关节自由, 不脱使能
6. stark_node 后启动时 stark_open 自动等待

## config.json

```json
{
  "motors": [
    { "id": 1, "auto_enable": false },
    { "id": 2, "auto_enable": false }
  ],
  "safety": {
    "heartbeat_timeout_ms": 1000
  }
}
```

- auto_enable=false: 算法控制使能时机 (外骨骼推荐)
- heartbeat_timeout_ms: 心跳超时后自动脱使能 (不发 FAULT)
